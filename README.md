# 🏥 Iraq PandemSim: Interactive Epidemic Intelligence System 🇮🇶

**Iraq PandemSim** is an integrated medical and epidemiological simulation system that bridges advanced statistical computing with interactive data visualization. The project leverages the statistical power of **R** and the flexible dashboard capabilities of **Python** to simulate and analyze disease spread across Iraq.

---

## 🚀 Features

- **Multi-Language Interoperability:** Uses **R** for heavy statistical analysis and data generation (exported as JSON), and **Python** to power the user-facing interactive web application.
- **Dynamic Medical Heatmap:** A spatial representation tracking infection cases across all 18 Iraqi governorates, dynamically responding to simulated lockdown and quarantine levels.
- **Decision-Making Balance:** A visual, data-driven analytical tool designed to balance economic impacts against projected lives saved during a health crisis.
- **Advanced Bio-Statistical Analysis:** High-quality medical and statistical charts generated using R's powerful `ggplot2` library.

---

## 🛠️ Tech Stack

- **Python:** `Streamlit` (UI Framework), `Folium` (Geospatial Mapping), `Pandas` (Data Manipulation).
- **R Language:** `ggplot2` (Advanced Visualization), `jsonlite` (Data Parsing & Structuring).
- **Data Pipeline:** `JSON` acts as the data bridge ensuring seamless cross-language communication between R and Python.

---

## 📂 Project Structure

- `epi_app.py` – The main web application interface (Python/Streamlit).
- `epi_model.py` – The core mathematical engine containing simulation logic and epidemiological algorithms.
- `iraq_analysis.R` – The R script responsible for data generation, preprocessing, and statistical plot creation.
- `cities_data.json` – The unified dataset generated by R, acting as the primary database for the Python frontend.
